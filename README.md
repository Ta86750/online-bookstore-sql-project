# 📚 Online Bookstore SQL Project

## 📌 Overview

This project demonstrates a relational database system for an online bookstore.
It uses real-world data from CSV files and performs various SQL operations for analysis.

---

## 🛠️ Technologies Used

* MySQL
* SQL

---

## 📂 Project Structure

* `schema.sql` → Database and table creation
* `data.sql` → Load data from CSV files
* `queries.sql` → SQL queries for analysis
* `data/` → CSV files (books, customers, orders)

---

## 🚀 How to Run

1. Run `schema.sql`
2. Import CSV data using `data.sql` or MySQL Import Wizard
3. Run `queries.sql`

---

## 🔥 Features

* Data import from CSV files
* Use of JOIN operations
* Aggregations (SUM, AVG, COUNT)
* Filtering, sorting, and grouping

---

## 💡 Key Queries

* Total revenue calculation
* Top-selling books
* Customer purchase analysis
* Genre-wise sales

---

## 📊 Learning Outcomes

* Hands-on SQL practice
* Working with relational databases
* Real-world data handling using CSV files
